Step to Open File:

1. Extract All Files.

2. In Command Line  (or other CLI) open the folder labeled capstone

3. Run command docker-compose up --build
 - this will start containers

4. To shut down the web application press ctrl and c on keyboard then enter docker-compose down in CLI
 - this will erase/loose any data that was entered into the datbase (mysql)

5. The Webpage names are below:
phpMyAdmin: localhost/8001
Soil Submission form: http://localhost/form_sub.html
Sign in:http://localhost/login.php

Database can only be accessed if signed in or registered.
Microplastic Databae: http://localhost/index.php
Microplatic Dashboard: http://localhost/dashboard.php
